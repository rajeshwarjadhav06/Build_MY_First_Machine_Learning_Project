# 🚀 First Machine Learning Project — KNN Model

## 📌 Project Overview
This project represents my first end-to-end Machine Learning journey where I built a model from scratch — starting from raw data to final evaluation.

Instead of only learning theory, I focused on implementing the complete ML pipeline to understand how real-world models are built and evaluated.

---

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess raw data
- Apply feature engineering techniques
- Build a Machine Learning model using KNN
- Evaluate model performance

---

## 🧠 End-to-End Workflow

### 🔹 Data Collection
- Loaded dataset
- Understood features and target variable

### 🔹 Data Cleaning
- Handled missing values
- Removed duplicates
- Fixed inconsistencies

### 🔹 Exploratory Data Analysis (EDA)
- Checked data distribution
- Analyzed correlations
- Detected outliers

### 🔹 Feature Engineering
- Encoded categorical variables
- Created new useful features
- Removed irrelevant columns

### 🔹 Feature Scaling
- Applied StandardScaler
- Ensured all features are on same scale

### 🔹 Model Building
- Implemented K-Nearest Neighbors (KNN)
- Tuned value of K for better accuracy

### 🔹 Model Evaluation
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

---

## 📊 Key Learnings
- Data preprocessing is more important than the algorithm
- Data quality directly impacts model performance
- Feature scaling is critical for KNN
- Debugging improves understanding more than theory
